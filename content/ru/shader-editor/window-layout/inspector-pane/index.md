Панель инспектора показывает детали текущего выбранного элемента.

Отображаемая информация зависит от типа выбранного актива.

Существуют инспекторы для следующих элементов:
- [Графы][1]
- [Узлы][2]
- [Материалы][3]
- [Текстуры][4]
