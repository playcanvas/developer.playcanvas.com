![ダッシュボード][1]

Project Dashboardはプロジェクトのコントロールセンターです。ダッシュボードから次のことができます：

* Access the Editor.
* Run the project's primary app.
* View the project's development log.
* View the project's activity.
* Edit your project's settings.
