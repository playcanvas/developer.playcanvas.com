--- title: components description: topic: playcanvas designer subtopic: components index: 3 --- # Components Components can be added to entities and provide some kind of specific functionality. For example, a camera component can be added to an entity to enable it to render the scene. The full list of available components is as follows:
ComponentDescription
animationSpecifies the animation assets that can run on the model specified by the entity's model component.
audiolistenerSpecifies the location of the listener for 3D audio playback.
audiosourceSpecifies audio assets that can be played at the position of the entity.
cameraRenders the scene from the location of the entity.
collisionboxSpecifies a box-shaped collision volume.
collisioncapsuleSpecifies a capsule-shaped collision volume.
collisionmeshSpecifies a collision shape based on the triangle mesh of a model asset.
collisionsphereSpecifies a sphere-shaped collision volume.
directionallightEnables the entity to emit directional light.
modelRenders a 3D model at the location of the entity.
pointlightEnables the entity to emit light from its position.
primitiveRenders a 3D graphics primitive as the location of the entity.
rigidbodyAdds the entity to the scene's physical simulation.
scriptAllows the script to run JavaScript fragments to implement custom behavior.
skyboxRenders a skybox in the scene.
spotlightEnables the entity to emit light from a spotlight cone.
